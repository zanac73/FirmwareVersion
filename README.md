# FirmwareVersion #

This is a simple library to serial print the firmware version 
with a predefined format.


## Examples ##


### Example 1 ###

```
FW Name: Firmware Version Demo
Version: 1.23 
Type: Development
Build date: 18:49:48 Feb  6 2024
```

### Example 2 ###

```
FW Name: Firmware Version Demo
Version: 1.23-rc
Type: ReleaseCandidate
Build date: 18:58:26 Feb  6 2024
```
